What is Angular 6?

Data Binding

Part of the power of Angular is it's ability to dynamically change data on a page as variables change. This is known as data binding.

Multi Platform

From mobile phones to laptops, Angular allows for any modern, evergreen web browser to load a powerful application without the need for any additional application installs.

Modularity

Modularity in programming is a life saver for most developers. It's no different in Angular. The way that a large scale website can be seperated into different modules helps keep an application readable, easy to extend and easy to implement.

Prerequisites

As a web based platform - any prior experience in web design will be useful for use in Angular. Any experience in object oriented programming will also be helpful when understanding the concepts of the language.
